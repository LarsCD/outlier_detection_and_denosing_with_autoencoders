# Project (NL)
## Uitleg
Dit project laat vormen van outlier detection en image de-nosing zien met behulp 
van autoencoders. Gebaseerd hierop zijn er 2 files die een kleine streamlit app vormen.

## Structuur
Het meeste van deze files zijn om het project modulair te houden. 
2 files zijn met ChatGPT veranderd in `.py` bestanden. 
Dus hoewel de `.py` files volledig met AI gegenereerd zijn, zijn de `.ipynb`'s waaruit ze zijn gegenereerd.

* `RUBEN Image outlier detection met de AutoEncoder - Les 3.ipynb` → `outlier_detection.py`
* `RUBEN Autoencoders image denoising - les 4.ipynb` → `denoising_autoencoder.py` + `blur_function.py`

## Streamlit
De files voor de streamlit app zijn `Outlier_(Streamlit).py` en `pages/Noise_(Streamlit).py`.
Deze files zorgen voor de functionaliteit van de website.

# Project (EN)  
## Explanation  
This project demonstrates forms of outlier detection and image denoising using autoencoders. Based on this, there are two files that together form a small Streamlit app.

## Structure  
Most of the files are used to keep the project modular.  
Two files were converted into `.py` scripts with the help of ChatGPT.  
Although the `.py` files were fully generated with AI, the original logic comes from the `.ipynb` notebooks they were created from.

* `RUBEN Image outlier detection met de AutoEncoder - Les 3.ipynb` → `outlier_detection.py`  
* `RUBEN Autoencoders image denoising - les 4.ipynb` → `denoising_autoencoder.py` + `blur_function.py`

## Streamlit  
The files for the Streamlit app are `Outlier_(Streamlit).py` and `pages/Noise_(Streamlit).py`. 
These files provide the functionality of the web interface.
